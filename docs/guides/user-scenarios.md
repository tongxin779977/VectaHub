# VectaHub 1.0.0 - 用户使用场景

## 前言
本文档收集了 VectaHub 的20个真实使用场景，供用户测试和参考。所有场景都是安全操作，无高危命令。

---

## 📋 使用场景列表

### 1️⃣ 项目启动检查
- **场景**: 刚打开 VectaHub，先检查环境是否正常
- **命令**: `vectahub doctor`
- **预期结果**: 显示所有检查项通过，确认 VectaHub 可以正常使用

---

### 2️⃣ 查看可用工具
- **场景**: 想知道 VectaHub 集成了哪些工具
- **命令**: `vectahub tools list`
- **预期结果**: 显示已注册的工具列表（git, npm, docker, curl）

---

### 3️⃣ 查看 Git 工具详细信息
- **场景**: 项目中使用 Git，想了解有哪些可用命令和限制
- **命令**: `vectahub tools info git`
- **预期结果**: 显示 Git 的描述、版本要求、命令列表和危险命令提示

---

### 4️⃣ 使用自然语言查看项目文件
- **场景**: 想快速看当前项目有哪些文件
- **命令**: `vectahub run "列出项目根目录文件"`
- **预期结果**: 执行意图匹配，生成并执行工作流

---

### 5️⃣ 查看 Git 状态
- **场景**: 想知道当前 Git 状态，有没有未提交的更改
- **命令**: `vectahub run "查看 git 状态"`
- **预期结果**: 匹配到 GIT_WORKFLOW 意图，执行 git status

---

### 6️⃣ 查看 npm 包信息
- **场景**: 想了解项目依赖包的情况
- **命令**: `vectahub run "查看 package.json"`
- **预期结果**: 显示 package.json 内容

---

### 7️⃣ 查看安全规则
- **场景**: 想了解 VectaHub 有哪些安全限制
- **命令**: `vectahub security list`
- **预期结果**: 显示所有安全规则列表

---

### 8️⃣ 测试某个命令是否安全
- **场景**: 想执行某个命令，先确认是否安全
- **命令**: `vectahub security test "git status"`
- **预期结果**: 显示命令安全级别（low/medium/high/critical）

---

### 9️⃣ 查看审计日志统计
- **场景**: 想看历史执行的统计
- **命令**: `vectahub audit stats`
- **预期结果**: 显示执行次数、成功/失败统计等

---

### 🔟 查询最近的审计日志
- **场景**: 查看最近执行了哪些操作
- **命令**: `vectahub audit query --limit 10`
- **预期结果**: 显示最近10条审计记录

---

### 1️⃣1️⃣ 查看当前执行模式
- **场景**: 了解当前是 strict/relaxed/consensus 模式
- **命令**: `vectahub mode`
- **预期结果**: 显示当前执行模式

---

### 1️⃣2️⃣ 查看项目开发状态
- **场景**: 检查项目开发进度
- **命令**: `vectahub dev status`
- **预期结果**: 显示各模块开发进度和完成度

---

### 1️⃣3️⃣ 检查项目配置
- **场景**: 确认项目配置是否正确
- **命令**: `vectahub dev check`
- **预期结果**: 检查 TypeScript、package.json、源目录等

---

### 1️⃣4️⃣ 列出当前保存的工作流
- **场景**: 看看有哪些可以复用的工作流
- **命令**: `vectahub list`
- **预期结果**: 显示已保存的工作流列表（如果有）

---

### 1️⃣5️⃣ 查看命令帮助
- **场景**: 不记得某个命令怎么用，看帮助

---

### 1️⃣6️⃣ 搜索相关工具
- **场景**: 想找跟网络相关的工具

### 1️⃣7️⃣ 查看 npm 工具的命令列表
- **场景**: 想知道 npm 工具支持哪些命令
- **命令**: `vectahub tools commands npm`
- **预期结果**: 显示 npm 工具的所有可用命令

---

### 1️⃣8️⃣ 查看执行历史
- **场景**: 想看看最近执行了哪些工作流
- **命令**: `vectahub history`
- **预期结果**: 显示最近的执行历史

---

### 1️⃣9️⃣ 使用自然语言查看 README
- **场景**: 想快速了解这个项目的说明
- **命令**: `vectahub run "查看 README 文件"`
- **预期结果**: 读取并显示 README.md 内容

---

### 2️⃣0️⃣ 查看完整帮助
- **场景**: 新手，想了解所有功能

- **预期结果**: 显示所有可用命令和选项
